진단부터 습관까지,
완성형 코칭 시스템

아이의 성향을 읽고, 매일을 관리하고, AI로 약점을 채우는 와와만의 자기주도학습 시스템.

4C PROCESS

검증된 4C 코칭 프로세스

진단 → 설계 → 관리 → 습관, 4단계로 자기주도학습을 완성합니다

1

Check · 진단

학습 성향, 집중력, 생활 패턴, 현재 학습 수준을 종합 진단합니다.

2

Custom · 설계

진단 결과를 바탕으로 아이 맞춤 학습 목표와 플랜을 설계합니다.

3

Coaching · 관리

전담 코치가 매일의 실행을 점검하고 1:1 피드백을 제공합니다.

4

Change · 습관

관리 없이도 스스로 공부하는 학습 습관으로 정착시킵니다.

맞춤 관리 코칭

아이마다 다른 학습 성향에 맞춰 학습 계획을 설계하고, 과목 코치가 매일 함께합니다. 관리받는 것을 넘어, 스스로 관리하는 법을 배웁니다.

  • ✓ 개인별 일일·주간 학습 플래너
  • ✓ 과목별 전문 코치 1:1 수업
  • ✓ 오답노트·취약 단원 집중 관리
  • ✓ 자기주도 학습 루틴 코칭
  • ✓ 학부모 주간 학습 리포트

AI 학습 클래스

AI가 아이의 학습 데이터를 분석해 어디서 막히는지 정확히 진단합니다. 취약 유형 맞춤 문제로 약점을 메우고, 코치가 그 과정을 함께 코칭합니다.

  • ✓ 단원별 이해도 자동 진단
  • ✓ 취약 유형 맞춤 문제 추천
  • ✓ 학습 데이터 기반 성장 그래프
  • ✓ AI 분석 + 코치 피드백 결합
